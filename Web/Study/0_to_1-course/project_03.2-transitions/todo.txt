Тех задание:
1) на странице должен быть подключен normalize
2) в стилях указана блочная модель border-box
3) Размеры секторов (больших квадратов) - 500px
4) Отступы между секторами (большими квадратами) 15px
5) Размеры квадратов с цифрами 250px
6) Для размещения квадратов с цифрами пользуемся позиционированием
7) Цифры в квадратах центрируем без позиционирования
8) Каждая из фигур, которая находится в одном секторе с квадратом, обозначенными цифрами,
является псевдоэлементом/-ами. То есть это не отдельный блок, а псевдоэлемент/-ы для квадрата с цифрой
9) При наведении на каждый из квадратов с цифрами псевдоэлемент,
который относится к этому квадрату должен меняться в соответствии со вторым изображением.
То есть при наведении на первый квадрат фигура в псевдоэлементе меняется на ту, что указана на втором изображении,
и сдвигается соответствующим образом
10) Для плавности смены цвета/перемещений и т.п. использовать свойство transition

Для определения цветов на картинке можно воспользоваться сайтом https://sanstv.ru/color
Треугольник можно сгенерировать через сервис http://apps.eky.hk/css-triangle-generator/

Задание не является обязательным, выполняется по желанию. Прислать можно в архиве с четвертым заданием
